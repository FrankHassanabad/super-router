    public addRoute(route: object): void source

    Adds a route to the router. Routes must be unique and deterministic (no splats or optionals). Routes can be annotated with additional information, will be available on the Request after matching.

    Any route parameters defined in the path will be available on the request.routeParams object when the handler is executed.

    If the route handler returns a promise, the app will wait for that promise to resolve or reject before moving to the next middleware in the stack.

    Params:

    NameTypeAttributeDescription
    route object

    The route definition
    route.path string

    The path pattern to be matched for this route
    route.method string

    The method that this route will match
    route.handler function

    The handler function to execute when the route is matched

    Return:

    void

    Example:

    router.addRoute({
 path : '/cases/:caseId',
 method : 'get',
 handler : ({request, response}) => {
   return Database.getCaseById(request.routeParams.caseId).then((caseInstance) => {
     request.setBody(caseInstance);
   });
 }
});

    public execute(options: object): void source

    Middleware to be consumed by a SuperRouter App instance. Causes the handler of a matched route to be executed. If request.matchedRoute is not defined, the middleware does nothing.

    Params:

    NameTypeAttributeDescription
    options object

    The request & response used by app middleware.

    Return:

    void

    Example:

    app.use(router.execute);

    public match(options: object): void source

    Middleware to be consumed by a SuperRouter App instance. Causes a request to be matched against the router. The matched route will be assigned to request.matchedRoute. In the case of no match, the middleware does not throw an error; the matchedRoute key will simply be undefined.

    Params:

    NameTypeAttributeDescription
    options object

    The request & response used by app middleware.

    Return:

    void

    Example:

    app.use(router.match);
